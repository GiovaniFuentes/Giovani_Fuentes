Focus App

A flexible productivity timer focused on simplicity, motivation, and stability.

← Back to Home

Project Overview

Focus App is a Pomodoro-inspired productivity application designed to help users manage focus sessions, breaks, and tasks without rigid time constraints. Instead of fixed cycles, the timer adapts to user behavior, rewarding progress with flexible break time and visual feedback.

Research Insights

  • Simplicity and clean UI strongly correlate with positive user reviews.
  • Users value customization without paywalls or unnecessary restrictions.
  • Gamification is motivating when subtle, but overwhelming when excessive.
  • Stability and performance issues are a major deal breaker.

Design Goals

  • Minimal onboarding with immediate usability
  • Flexible work and break timing
  • Integrated task and subtask management
  • Lightweight gamification to encourage consistency

Design Process & Iteration

The project began with low-fidelity sketches to explore layout and hierarchy before functionality. The primary goal was to test whether users could understand the app’s purpose at a glance.

Initial Concept

Low-Fidelity Prototypes

Two “Wizard of Oz” low-fidelity prototypes were created to evaluate clarity, navigation, and perceived complexity without full functionality.

Prototype 1 — Radial Timer

Prototype 1 emphasized simplicity with a large radial timer and minimal text. Users consistently found this version easier to understand and less intimidating.

Prototype 2 — Expanded Controls

Prototype 2 added task completion and additional controls. While users appreciated the functionality, many felt the interface was more cluttered and required additional explanation.

User Testing Results

  • Prototype 1 was preferred for clarity and ease of use
  • Prototype 2 provided useful features but felt overwhelming
  • Users wanted clearer labels, feedback, and onboarding
  • Visual and auditory feedback increased motivation

Final Direction

The final design primarily builds on Prototype 1, incorporating selective features from Prototype 2. The result is a clean, approachable interface with customizable settings, task tracking, and subtle motivational feedback.
